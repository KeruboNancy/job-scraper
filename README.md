# Job Scraper (Python)

This is a simple Python project that scrapes job listings from a website and saves them into a CSV file.

## Features

- Extracts job title, company name, and application link
- Saves data into a structured CSV file
- Uses requests and BeautifulSoup for web scraping

## Technologies Used

- Python
- requests
- BeautifulSoup

## How It Works

1. Sends a request to the website
2. Parses the HTML content
3. Extracts job data
4. Saves the data into a CSV file

## How to Run

```bash
pip install requests beautifulsoup4
python Web Scraper.py
```

## Output

A file named jobs.csv containing job listings.
